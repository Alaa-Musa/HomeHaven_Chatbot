# Smart Home

## 1) SecureNest Smart Lock (Model: SN-L2)
**Price:** AED 549  
**Unlock methods:**
- PIN code
- Fingerprint
- Mobile app (Bluetooth)
- Physical key backup

**Door compatibility (summary):**
- Door thickness: 35–55 mm
- Supports common deadbolt prep (see HH-11 for exact requirements)

**Power:** 4x AA batteries (typical 6–9 months)  
**Installation:** Available as add-on (AED 99) — see HH-11
