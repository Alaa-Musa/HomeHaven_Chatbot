# SecureNest SN-L2 — Installation Guide (Compatibility + Setup)

## A) Door compatibility requirements (important)
- Door thickness: **35–55 mm**
- Backset: **60 mm or 70 mm**
- Existing deadbolt hole: **54 mm** diameter (common standard)
- Edge bore: **25 mm** diameter
- Works best with a single-cylinder deadbolt door prep

If your door does not match these measurements, installation may require additional drilling.

## B) HomeHaven installation add-on
**Fee:** AED 99  
Includes:
- Removing existing lock (if compatible)
- Fitting SN-L2 exterior and interior units
- Deadbolt alignment check
- Basic app pairing (Bluetooth) and PIN setup

Not included:
- Painting/repairing door damage
- Network/Wi-Fi troubleshooting (SN-L2 uses Bluetooth for app access)
- Advanced smart-home integrations

## C) Battery and maintenance
- Uses **4x AA batteries**
- Low battery warning appears in-app and on the keypad
- Keep the backup key accessible outside the home (trusted person recommended)

## D) Initial setup steps (customer-friendly)
1. Insert batteries
2. Create admin PIN
3. Add fingerprints (up to the device limit shown in the app)
4. Test lock/unlock with door open first
5. Calibrate latch throw (follow in-app prompts)
