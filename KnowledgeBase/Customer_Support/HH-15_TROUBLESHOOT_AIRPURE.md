# AirPure AP-400 — Common Issues

## 1) Unit won’t turn on
- Confirm power cable is fully inserted
- Try another outlet
- If using a power strip, test without it

## 2) Weak airflow
- Check if the filter is still in plastic packaging (first-time setup mistake)
- Clean pre-filter (vacuum gently)
- Replace main filter if “Filter” indicator persists

## 3) Strange smell
- New unit smell can occur for the first few hours
- If smell persists: replace carbon filter layer (if included in your filter pack)

## 4) Loud noise
- Ensure purifier is on a flat surface
- Check for foreign objects near the fan intake
- Use Sleep mode at night

## 5) Filter replacement note
AP-400 uses replacement pack **AP-F400** (sold separately).  
Filter life varies by air quality; heavy dust environments shorten lifespan.
